# Library API

## Video
Link: (https://youtu.be/oTo7qOJikiA)

## Deskripsi
REST API untuk mengelola data buku.

## Teknologi
- ASP.NET Core
- PostgreSQL
- Dapper

## Cara Menjalankan
1. Import database.sql
2. Jalankan project
3. Buka /swagger

## Endpoint
GET /api/books  
POST /api/books  
PUT /api/books/{id}  
DELETE /api/books/{id}
